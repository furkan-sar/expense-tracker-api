# Expense Tracker API

Backend API for personal and shared budget management.

## Features

* User registration
* User authentication
* Personal budgets
* Shared budgets
* Budget members
* Categories
* Transactions
* Reports
* Cloud synchronization

## Tech Stack

* Java 21
* Spring Boot
* Maven
* PostgreSQL
* Spring Security
* JWT
* Spring Data JPA
* Validation
* MapStruct
* Lombok

## Architecture

Layered Architecture:

* Controller
* Service
* Repository
* Entity
* DTO
* Mapper

## API Versioning

All endpoints must use:

/api/v1

## Source Of Truth

API contract is located at:

contracts/openapi/expense-tracker-api.yaml
